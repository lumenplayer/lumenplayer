## 初始設定

### 新增影片源

首次啟動 Lumen 後，需要新增至少一個影片源才能開始瀏覽。

1. 開啟 **設定**（選單列 → Lumen → 設定，或快捷鍵 `⌘ ,`）
2. 切換到 **影片源** 標籤頁
3. 點擊右上角 **「新增影片源」** 按鈕
4. 在彈窗中填寫：
   - **影片源名稱**（必填）：為這個源取一個名字，方便識別
   - **URL 或口令**：輸入訂閱源的 HTTPS 位址，或直接輸入口令（如 `alianglaile`）
5. 點擊 **「新增」**

> **💡 提示**：口令是一種快捷方式。例如輸入 `alianglaile`，Lumen 會自動解析為對應的訂閱源 URL，無需手動查找位址。

新增成功後，Lumen 會自動下載並解析配置，側邊欄將顯示目前站點名稱。

### 多源管理

- 你可以新增多個影片源，每個源包含若干個站點
- 在設定頁面中，可以對每個源進行 **啟用/停用**、**編輯**、**刪除** 操作
- 源內的站點也可以單獨啟用或停用
- 點擊 🔄 按鈕可以偵測源內各站點的可用性（綠色 = 正常，紅色 = 異常）

## 側邊欄導覽

Lumen macOS 版採用經典的側邊欄 + 內容區佈局。

側邊欄包含以下導覽項：

| 導覽項 | 圖示 | 功能 |
|--------|------|------|
| **站點名稱** | — | 顯示目前站點，點擊彈出站點切換器 |
| **搜尋** | 🔍 | 關鍵詞搜尋及分類瀏覽 |
| **主頁** | 🏠 | 目前站點的推薦內容和分類 |
| **播放歷史** | 🕒 | 觀看記錄和續播入口 |
| **我的收藏** | ❤️ | 已收藏的影片列表 |

### 切換站點

1. 點擊側邊欄頂部的站點名稱
2. 在彈出的站點列表中選擇目標站點
3. 主頁和搜尋頁內容會自動重新整理為新站點的資料

## 主頁

主頁是進入 Lumen 後的預設內容頁面，展示目前站點的推薦內容。

### Hero Banner 推薦輪播

- 頁面頂部顯示全幅 Hero Banner，自動輪播推薦影片
- 每張卡片展示影片海報、名稱等資訊
- 點擊 Banner 卡片可直接進入影片詳情頁

### 繼續播放

- 如果有未看完的影片，主頁會顯示「繼續播放」行
- 每個卡片顯示影片海報和播放進度條
- 點擊即可從上次位置繼續播放，無需手動尋找
- 如果對應站點已失效，會自動搜尋其他可用站點進行續播

### 我的收藏

- 主頁也會展示最近收藏的影片
- 點擊收藏卡片進入影片詳情頁

### 分類內容

- 主頁下方展示目前站點的各分類內容（如電影、電視劇、綜藝、動漫等）
- 每個分類以橫向捲動行展示，可以滑鼠滾輪或觸控板左右滑動
- 點擊行標題右側的 **「更多」** 可跳轉到搜尋頁的對應分類列表
- 點擊影片卡片進入詳情頁

### 主題源說明

如果目前站點是 **主題源**，它只提供影片推薦資訊，不直接提供播放連結。點擊主題源的影片卡片後，Lumen 會自動搜尋其他可播放的站點取得播放連結。

```
主頁 → 點擊主題源推薦影片 → 自動搜尋可播放站點 → 詳情頁 → 播放
```

## 搜尋頁

搜尋頁是 Lumen 使用頻率最高的頁面，整合了**關鍵詞搜尋**和**分類瀏覽**兩大功能。

### 搜尋框

- 位於頁面頂部工具列中
- 輸入關鍵詞後按 **Return** 發起搜尋
- 點擊搜尋框右側的 ✕ 按鈕可清除搜尋內容並返回瀏覽介面

### 搜尋歷史

- 在未輸入搜尋詞時，頁面頂部顯示「最近搜尋」橫向捲動行
- 每個歷史卡片展示搜尋詞和封面縮圖
- 點擊任意歷史卡片可直接重新發起該搜尋
- 點擊右上角 **「清除」** 可清空所有搜尋歷史

---

### 分類瀏覽

搜尋頁下方展示目前站點的分類卡片牆。

#### 分類卡片牆

- 每個分類以海報封面卡片展示（如「電影」「電視劇」「綜藝」「動漫」等）
- 卡片使用漸層色區分不同分類，滑鼠懸停時有縮放和高亮動效
- 點擊分類卡片可進入該分類的內容列表

#### 分類內容列表

進入分類後，頁面內容切換為該分類的影片列表：

1. **頂部顯示分類名稱**，左上角出現返回按鈕
2. **標籤篩選欄**（如果分類配置了標籤）：
   - 標籤橫向排列，如「全部」「大陸」「港台」「日韓」「歐美」等
   - 點擊標籤可篩選對應內容
3. **影片網格**：
   - 以海報卡片網格展示該分類下的所有影片
   - 每張卡片顯示封面、名稱和評分
   - 捲動到底部自動載入更多
   - 也可以點擊「載入更多」按鈕手動載入
4. **點擊影片卡片** → 進入詳情頁

```
搜尋頁 → 分類卡片牆 → 點擊分類 → 標籤篩選 → 瀏覽影片列表 → 點擊影片 → 詳情頁 → 播放
```

---

### 關鍵詞搜尋

1. 在搜尋框中輸入影片名稱或關鍵詞
2. 按 **Return** 發起搜尋
3. Lumen 會同時向所有已啟用且支援搜尋的站點發出請求
4. 搜尋過程中顯示載入指示器，結果會即時追加到頁面
5. 結果以海報卡片網格展示，顯示影片總數
6. 每張卡片顯示所屬站點名稱，方便區分不同來源
7. 點擊影片卡片進入詳情頁

```
搜尋頁 → 輸入關鍵詞 → Return 搜尋 → 搜尋結果網格 → 點擊影片 → 詳情頁 → 播放
```

## 詳情頁

詳情頁展示影片的完整資訊和播放入口，採用 Apple TV 風格的沉浸式設計。

### 頂部 Hero 區域

- **全幅背景圖**：優先顯示橫幅劇照，無劇照時自動模糊封面圖替代
- **影片標題**：大號粗體字，帶陰影效果
- **元資料徽章**：以膠囊形徽章展示類型、年份、地區、評分
- **收藏按鈕**：點擊 ❤️ 徽章可收藏/取消收藏影片
- **簡介**：影片描述，點擊「更多」可展開完整資訊彈窗
- **演職人員**：右側顯示導演和主演資訊

### 播放源切換

- 如果影片有多個播放源（Flag），頁面會顯示「播放源」切換列
- 以按鈕組形式展示所有可用源（如「量子」「光速」「閃電」等）
- 點擊不同源名稱切換劇集列表

### 劇集列表

影片的劇集列表根據內容自動選擇展示方式：

#### 卡片式劇集（帶縮圖）

- 當劇集有縮圖時，以橫向捲動的大卡片展示
- 每張卡片包含：劇集縮圖、集數標籤、劇集名稱、簡介
- 目前播放集高亮顯示白色邊框

#### 緊湊式劇集網格

- 當劇集無縮圖時，以緊湊網格展示
- 每個按鈕顯示劇集名稱
- 目前播放集以品牌色高亮並顯示播放圖示
- 按鈕使用液體玻璃效果

### 播放操作

1. 點擊任意劇集開始播放
2. 播放前會顯示「正在解析播放位址…」載入狀態
3. 解析成功後自動進入播放器

### 換源結果

- 頁面底部展示「換源結果」區域
- 自動搜尋其他站點中的同名影片
- 以橫向捲動的海報卡片展示，顯示所屬站點名稱
- 點擊換源卡片可切換到該站點的影片詳情和劇集

> **💡 使用情境**：當某個站點的播放連結失效時，可以透過換源結果找到其他可用站點。

## 播放器

點擊劇集後進入全螢幕播放器。

### 播放控制

- **播放/暫停**：點擊螢幕中央或按 `空白鍵`
- **快進/快退**：按方向鍵 `←` `→`（預設 10 秒，可在設定中調整）
- **進度拖動**：拖動底部進度條
- **音量控制**：使用系統音量鍵
- **倍速播放**：支援 0.5x ~ 2.0x
- **子母畫面**：點擊子母畫面按鈕，邊看邊做其他事

### 自動續播

- 每次觀看的進度自動儲存
- 下次開啟同一集會從上次位置繼續播放

### 自動播放下一集

- 目前集播放完畢後自動播放下一集（可在設定中關閉）

## 播放歷史

播放歷史頁面記錄所有觀看過的影片。

### 歷史記錄卡片

- 以 5 欄海報網格展示
- 每張卡片顯示：
  - 影片封面圖
  - **播放進度條**（底部藍色進度條，直觀顯示觀看進度）
  - 影片名稱
  - 站點名稱和集數資訊

### 一鍵續播

- 點擊任意歷史卡片即可從上次進度繼續播放
- 載入期間卡片會顯示載入遮罩
- 如果原始站點失效，Lumen 會自動搜尋其他可用站點

### 管理歷史

- **右鍵單條刪除**：右鍵點擊歷史卡片 → 選取「刪除」
- **清空全部**：點擊頁面右上角「清空歷史」按鈕，需二次確認

```
播放歷史 → 點擊歷史卡片 → 自動續播（從上次進度繼續）
```

## 我的收藏

收藏頁面管理所有已收藏的影片。

### 收藏列表

- 以 5 欄海報網格展示
- 顯示影片封面、名稱和評分徽章
- 頂部顯示收藏總數

### 排序方式

- **收藏時間**（預設）：按收藏時間倒序排列
- **名稱**：按影片名稱字母序排列
- 透過右上角的排序選擇器切換

### 操作

- **進入詳情**：點擊收藏卡片 → 進入影片詳情頁 → 選集播放
- **取消收藏**：右鍵點擊收藏卡片 → 選取「取消收藏」

### 如何收藏影片

在影片詳情頁，點擊元資料徽章行中的 **❤️ 收藏** 按鈕即可。收藏後可透過 iCloud 同步到其他裝置。

```
我的收藏 → 點擊收藏影片 → 詳情頁 → 選集播放
```

## 設定

透過選單列 → Lumen → 設定（`⌘ ,`）開啟設定視窗。

### 一般設定

| 選項 | 說明 | 預設值 |
|------|------|--------|
| 預設倍速 | 播放時的預設播放速度 | 1.0x |
| 快進/快退步長 | 方向鍵快進快退的秒數 | 10 秒 |
| 自動播放下一集 | 播完一集後自動播放下一集 | 開啟 |
| 外觀模式 | 跟隨系統 / 深色 / 淺色 | 跟隨系統 |

### 影片源管理

影片源標籤頁提供完整的源管理功能：

| 操作 | 說明 |
|------|------|
| ➕ **新增** | 透過 URL 或口令新增影片源 |
| ✏️ **編輯** | 修改源的名稱或 URL |
| 🗑️ **刪除** | 刪除影片源及其所有關聯站點 |
| 🔄 **偵測** | 偵測源內各站點的可用性 |
| 🔒 **加密** | 為影片源設定密碼保護 |
| 🔛 **啟用/停用** | 切換源或站點的啟用狀態 |

每個源展開後會顯示其包含的所有站點列表，包括：
- 站點名稱、優先順序、類型（外掛/直連/主題）
- 偵測狀態（綠/紅指示燈）
- 備註資訊
- 單站點啟用/停用開關

### 進階設定

進階設定提供細粒度的快取時間控制：

| 快取類型 | 說明 | 預設值 |
|----------|------|--------|
| 主頁快取 | 主頁內容快取時長 | 4 小時 |
| 主頁焦點圖快取 | Hero Banner 快取時長 | 48 小時 |
| 分類列表快取 | 分類列表快取時長 | 7 天 |
| 分類內容快取 | 分類內的影片列表快取 | 4 小時 |
| 詳情頁快取 | 影片詳情頁快取 | 1 小時 |
| 完結內容快取 | 已完結影片的快取 | 7 天 |
| 外掛引擎快取 | 外掛腳本快取 | 4 小時 |

- 點擊 **「清除全部快取」** 可釋放所有快取佔用的儲存空間
- 快取時間修改僅對新寫入的快取生效

## 鍵盤快捷鍵

| 快捷鍵 | 功能 |
|--------|------|
| `⌘ ,` | 開啟設定 |
| `空白鍵` | 播放 / 暫停 |
| `←` `→` | 快退 / 快進（預設 10 秒） |
| `Return` | 發起搜尋 |
| `Esc` | 退出播放器 |
