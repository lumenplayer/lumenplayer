# Lumen Player

[简体中文](README-cn.md) | [English](README-en.md)

> 你的所有影視，盡在一個應用。

Lumen 是一款跨平台影視點播應用，透過配置驅動的方式聚合多個影視站點，支援 macOS、Apple TV、iPhone 和 iPad。

---

## ✨ 核心特性

| 特性 | 說明 |
|------|------|
| 🎬 **多源聚合** | 匯入訂閱源，將數十個影視站點整合到一個應用 |
| 🔍 **智慧搜尋** | 輸入關鍵詞，同時搜尋所有已配置的影片源 |
| ▶️ **一鍵播放** | 自動解析播放連結，智慧匹配最佳線路 |
| ☁️ **iCloud 同步** | 播放進度、收藏列表跨裝置自動同步 |
| 🧩 **外掛系統** | 支援 LumenPlugin 和 MacCMS 直連 API 兩種影片源類型 |
| 🔒 **密碼保護** | 影片源支援密碼加密，保護私人配置 |
| ⌨️ **快捷鍵** | macOS 上全覆蓋的鍵盤快捷鍵操作 |
| 📡 **AirPlay** | 將畫面投射到大螢幕電視 |

## 🚀 快速開始

### 第一步：新增影片源

開啟 Lumen → 設定（`⌘ ,`）→ **影片源** 標籤頁 → **新增影片源**。

#### 方式一：透過 URL 匯入

在彈窗中填寫影片源名稱和訂閱源 JSON 位址（須以 `https://` 開頭），點擊新增。

#### 方式二：透過口令匯入

在 URL 框中輸入口令（如 `alianglaile`），應用會自動識別並解析為對應的訂閱源。

### 第二步：開始瀏覽

新增完成後，應用會自動載入影片源。側邊欄顯示目前站點名稱，點擊可切換站點。

## 📺 推薦訂閱源

> 以下是社群維護的第三方訂閱源，僅供參考。源的可用性取決於網路環境和維護者更新情況。

| 維護者 | 源名稱 | 專案位址 / 口令 | 備註 |
|--------|--------|----------------|------|
| **alianglaile** | 測試源 | 口令 `alianglaile` | 包含多個常用站點 |
| **alianglaile** | 大陸測試源 | 口令 `alianglailecn` | 包含多個大陸可用站點 |
| **alianglaile** | 18+ 測試源 | 口令 `alianglaile18` | 包含多個成人站點 |
| **Biu** | 付費源 | [詳情](https://info.infiniteapi.com/about) | 需付費 |


> [!IMPORTANT]
> 建議不要新增過多的影視站點，**3–5 個高品質站點足矣**。過多的站點會顯著拖慢搜尋速度，因為搜尋時會同時請求所有已啟用的站點。對於不用的站點，可以在影片源設定中單獨停用。

> [!TIP]
> 🔌 **歡迎貢獻外掛！** 編寫外掛可參考：[lumenplayer/lumenplugin](https://github.com/lumenplayer/lumenplugin)
>
> Lumen 相容 [SyncNext](https://github.com/qoli/syncnext-api) 影片源格式。

## 🎯 典型使用流程

### 流程一：分類瀏覽
```
搜尋頁 → 分類卡片牆 → 選擇分類 → 瀏覽列表（翻頁/篩選）→ 點擊影片 → 詳情頁 → 選集播放
```

### 流程二：關鍵詞搜尋
```
搜尋頁 → 輸入關鍵詞 → 搜尋結果 → 點擊影片 → 詳情頁 → 選集播放
```

### 流程三：首頁推薦
```
首頁 → 熱門劇集/電影/綜藝 → 點擊推薦卡片 → 詳情頁 → 選集播放
```

### 流程四：續播
```
播放歷史 → 點擊歷史記錄 → 自動續播（從上次進度繼續）
```

### 流程五：收藏回看
```
我的收藏 → 點擊收藏影片 → 詳情頁 → 選集播放
```

## ❓ 常見問題

### 搜尋無結果？
- 確認影片源的 `search` 欄位為 `true`
- 部分站點有 IP/地區限制
- 嘗試搜尋熱門影片名稱排除源本身故障

### 可搜尋但無法播放？
- `theme` 類型的源僅提供推薦，點擊後會自動搜尋其他可播放源
- 部分 VIP 內容可能被源端過濾
- 某些站點需要特定地區 IP

### 訂閱源載入失敗？
- 檢查訂閱 URL 是否可存取
- 確認 JSON 格式正確（支援帶註解和尾逗號的非標準 JSON）
- 檢查網路代理設定
