# Lumen Player

> 你的所有影视，尽在一个应用。

Lumen 是一款跨平台影视点播应用，通过配置驱动的方式聚合多个影视站点，支持 macOS、Apple TV、iPhone 和 iPad。

---

## ✨ 核心特性

| 特性 | 说明 |
|------|------|
| 🎬 **多源聚合** | 导入订阅源，将数十个影视站点整合到一个应用 |
| 🔍 **智能搜索** | 输入关键词，同时搜索所有已配置的视频源 |
| ▶️ **一键播放** | 自动解析播放链接，智能匹配最佳线路 |
| ☁️ **iCloud 同步** | 播放进度、收藏列表跨设备自动同步 |
| 🧩 **插件系统** | 支持 LumenPlugin 和 MacCMS 直连 API 两种视频源类型 |
| 🔒 **密码保护** | 视频源支持密码加密，保护私人配置 |
| ⌨️ **快捷键** | macOS 上全覆盖的键盘快捷键操作 |
| 📡 **AirPlay** | 将画面投射到大屏电视 |

## 🚀 快速开始

### 第一步：添加视频源

打开 Lumen → 设置（`⌘ ,`）→ **视频源** 标签页 → **新增视频源**。

#### 方式一：通过 URL 导入

在弹窗中填写视频源名称和订阅源 JSON 地址（须以 `https://` 开头），点击添加。

#### 方式二：通过口令导入

在 URL 框中输入口令（如 `alianglaile`），应用会自动识别并解析为对应的订阅源。

### 第二步：开始浏览

添加完成后，应用会自动加载视频源。侧边栏显示当前站点名称，点击可切换站点。

> [!IMPORTANT]

## 📖 页面概览

| 页面 | 功能 |
|------|------|
| 🏠 **主页** | Hero Banner 推荐轮播、继续播放、我的收藏、站点分类内容 |
| 🔎 **搜索** | 关键词搜索 + 分类浏览（含标签筛选），搜索历史记录 |

| ❤️ **我的收藏** | 收藏列表管理，快速进入详情页 |
| ⚙️ **设置** | 视频源管理、播放参数、缓存设置 |

## 🎯 典型使用流程

### 流程一：分类浏览
```
搜索页 → 分类卡片墙 → 选择分类 → 浏览列表（翻页/筛选）→ 点击影片 → 详情页 → 选集播放
```

### 流程二：关键词搜索
```
搜索页 → 输入关键词 → 搜索结果 → 点击影片 → 详情页 → 选集播放
```

### 流程三：主页推荐
```
主页 → 热门剧集/电影/综艺 → 点击推荐卡片 → 详情页 → 选集播放
```

### 流程四：续播
```
播放历史 → 点击历史记录 → 自动续播（从上次进度继续）
```

### 流程五：收藏回看
```
我的收藏 → 点击收藏影片 → 详情页 → 选集播放
```

## ❓ 常见问题

### 搜索无结果？
- 确认视频源的 `search` 字段为 `true`
- 部分站点有 IP/地区限制
- 尝试搜索热门影片名称排除源本身故障

### 可搜索但无法播放？
- `theme` 类型的源仅提供推荐，点击后会自动搜索其他可播放源
- 部分 VIP 内容可能被源端过滤
- 某些站点需要特定地区 IP

### 订阅源加载失败？
- 检查订阅 URL 是否可访问
- 确认 JSON 格式正确（支持带注释和尾逗号的非标准 JSON）
- 检查网络代理设置
