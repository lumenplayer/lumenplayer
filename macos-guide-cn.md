## 初始设置

### 添加视频源

首次启动 Lumen 后，需要添加至少一个视频源才能开始浏览。

1. 打开 **设置**（菜单栏 → Lumen → 设置，或快捷键 `⌘ ,`）
2. 切换到 **视频源** 标签页
3. 点击右上角 **「新增视频源」** 按钮
4. 在弹窗中填写：
   - **视频源名称**（必填）：为这个源起一个名字，方便识别
   - **URL 或口令**：输入订阅源的 HTTPS 地址，或直接输入口令（如 `alianglaile`）
5. 点击 **「添加」**

> **💡 提示**：口令是一种快捷方式。例如输入 `alianglaile`，Lumen 会自动解析为对应的订阅源 URL，无需手动查找地址。

添加成功后，Lumen 会自动下载并解析配置，侧边栏将显示当前站点名称。

### 多源管理

- 你可以添加多个视频源，每个源包含若干个站点
- 在设置页面中，可以对每个源进行 **启用/禁用**、**编辑**、**删除** 操作
- 源内的站点也可以单独启用或禁用
- 点击 🔄 按钮可以检测源内各站点的可用性（绿色 = 正常，红色 = 异常）

## 侧边栏导航

Lumen macOS 版采用经典的侧边栏 + 内容区布局。

侧边栏包含以下导航项：

| 导航项 | 图标 | 功能 |
|--------|------|------|
| **站点名称** | — | 显示当前站点，点击弹出站点切换器 |
| **搜索** | 🔍 | 关键词搜索及分类浏览 |
| **主页** | 🏠 | 当前站点的推荐内容和分类 |
| **播放历史** | 🕒 | 观看记录和续播入口 |
| **我的收藏** | ❤️ | 已收藏的影片列表 |

### 切换站点

1. 点击侧边栏顶部的站点名称
2. 在弹出的站点列表中选择目标站点
3. 主页和搜索页内容会自动刷新为新站点的数据

## 主页

主页是进入 Lumen 后的默认内容页面，展示当前站点的推荐内容。

### Hero Banner 推荐轮播

- 页面顶部显示全幅 Hero Banner，自动轮播推荐影片
- 每张卡片展示影片海报、名称等信息
- 点击 Banner 卡片可直接进入影片详情页

### 继续播放

- 如果有未看完的影片，主页会显示「继续播放」行
- 每个卡片显示影片海报和播放进度条
- 点击即可从上次位置继续播放，无需手动寻找
- 如果对应站点已失效，会自动搜索其他可用站点进行续播

### 我的收藏

- 主页也会展示最近收藏的影片
- 点击收藏卡片进入影片详情页

### 分类内容

- 主页下方展示当前站点的各分类内容（如电影、电视剧、综艺、动漫等）
- 每个分类以横向滚动行展示，可以鼠标滚轮或触控板左右滑动
- 点击行标题右侧的 **「更多」** 可跳转到搜索页的对应分类列表
- 点击影片卡片进入详情页

### 主题源说明

如果当前站点是 **主题源**，它只提供影片推荐信息，不直接提供播放链接。点击主题源的影片卡片后，Lumen 会自动搜索其他可播放的站点获取播放链接。

```
主页 → 点击主题源推荐影片 → 自动搜索可播放站点 → 详情页 → 播放
```

## 搜索页

搜索页是 Lumen 使用频率最高的页面，集成了**关键词搜索**和**分类浏览**两大功能。

### 搜索框

- 位于页面顶部工具栏中，采用胶囊形液体玻璃样式
- 输入关键词后按 **回车** 发起搜索
- 点击搜索框右侧的 ✕ 按钮可清除搜索内容并返回浏览界面

### 搜索历史

- 在未输入搜索词时，页面顶部显示「最近搜索」横向滚动行
- 每个历史卡片展示搜索词和封面缩略图
- 点击任意历史卡片可直接重新发起该搜索
- 点击右上角 **「清除」** 可清空所有搜索历史

---

### 分类浏览

搜索页下方展示当前站点的分类卡片墙。

#### 分类卡片墙

- 每个分类以海报封面卡片展示（如「电影」「电视剧」「综艺」「动漫」等）
- 卡片使用渐变色区分不同分类，鼠标悬停时有缩放和高亮动效
- 点击分类卡片可进入该分类的内容列表

#### 分类内容列表

进入分类后，页面内容切换为该分类的影片列表：

1. **顶部显示分类名称**，左上角出现返回按钮
2. **标签筛选栏**（如果分类配置了标签）：
   - 标签横向排列，如「全部」「大陆」「港台」「日韩」「欧美」等
   - 点击标签可筛选对应内容
3. **影片网格**：
   - 以海报卡片网格展示该分类下的所有影片
   - 每张卡片显示封面、名称和评分
   - 滚动到底部自动加载更多
   - 也可以点击「加载更多」按钮手动加载
4. **点击影片卡片** → 进入详情页

```
搜索页 → 分类卡片墙 → 点击分类 → 标签筛选 → 浏览影片列表 → 点击影片 → 详情页 → 播放
```

---

### 关键词搜索

1. 在搜索框中输入影片名称或关键词
2. 按 **回车** 发起搜索
3. Lumen 会同时向所有已启用且支持搜索的站点发出请求
4. 搜索过程中显示加载指示器，结果会实时追加到页面
5. 结果以海报卡片网格展示，显示影片总数
6. 每张卡片显示所属站点名称，方便区分不同来源
7. 点击影片卡片进入详情页

```
搜索页 → 输入关键词 → 回车搜索 → 搜索结果网格 → 点击影片 → 详情页 → 播放
```

## 详情页

详情页展示影片的完整信息和播放入口，采用 Apple TV 风格的沉浸式设计。

### 顶部 Hero 区域

- **全幅背景图**：优先显示横幅剧照，无剧照时自动模糊封面图替代
- **影片标题**：大号粗体字，带阴影效果
- **元数据徽章**：以胶囊形徽章展示类型、年份、地区、评分
- **收藏按钮**：点击 ❤️ 徽章可收藏/取消收藏影片
- **简介**：影片描述，点击「更多」可展开完整信息弹窗
- **演职人员**：右侧显示导演和主演信息

### 播放源切换

- 如果影片有多个播放源（Flag），页面会显示「播放源」切换栏
- 以按钮组形式展示所有可用源（如「量子」「光速」「闪电」等）
- 点击不同源名称切换剧集列表

### 剧集列表

影片的剧集列表根据内容自动选择展示方式：

#### 卡片式剧集（带缩略图）

- 当剧集有缩略图时，以横向滚动的大卡片展示
- 每张卡片包含：剧集缩略图、集数标签、剧集名称、简介
- 当前播放集高亮显示白色边框

#### 紧凑式剧集网格

- 当剧集无缩略图时，以紧凑网格展示
- 每个按钮显示剧集名称
- 当前播放集以品牌色高亮并显示播放图标
- 按钮使用液体玻璃效果

### 播放操作

1. 点击任意剧集开始播放
2. 播放前会显示「正在解析播放地址…」加载状态
3. 解析成功后自动进入播放器

### 换源结果

- 页面底部展示「换源结果」区域
- 自动搜索其他站点中的同名影片
- 以横向滚动的海报卡片展示，显示所属站点名称
- 点击换源卡片可切换到该站点的影片详情和剧集

> **💡 使用场景**：当某个站点的播放链接失效时，可以通过换源结果找到其他可用站点。

## 播放器

点击剧集后进入全屏播放器。

### 播放控制

- **播放/暂停**：点击屏幕中央或按 `空格键`
- **快进/快退**：按方向键 `←` `→`（默认 10 秒，可在设置中调整）
- **进度拖动**：拖动底部进度条
- **音量控制**：使用系统音量键
- **倍速播放**：支持 0.5x ~ 2.0x
- **画中画**：点击画中画按钮，边看边做其他事

### 自动续播

- 每次观看的进度自动保存
- 下次打开同一集会从上次位置继续播放

### 自动播放下一集

- 当前集播放完毕后自动播放下一集（可在设置中关闭）

## 播放历史

播放历史页面记录所有观看过的影片。

### 历史记录卡片

- 以 5 列海报网格展示
- 每张卡片显示：
  - 影片封面图
  - **播放进度条**（底部蓝色进度条，直观显示观看进度）
  - 影片名称
  - 站点名称和集数信息

### 一键续播

- 点击任意历史卡片即可从上次进度继续播放
- 加载期间卡片会显示加载遮罩
- 如果原始站点失效，Lumen 会自动搜索其他可用站点

### 管理历史

- **右键单条删除**：右键点击历史卡片 → 选择「删除」
- **清空全部**：点击页面右上角「清空历史」按钮，需二次确认

```
播放历史 → 点击历史卡片 → 自动续播（从上次进度继续）
```

## 我的收藏

收藏页面管理所有已收藏的影片。

### 收藏列表

- 以 5 列海报网格展示
- 显示影片封面、名称和评分徽章
- 顶部显示收藏总数

### 排序方式

- **收藏时间**（默认）：按收藏时间倒序排列
- **名称**：按影片名称字母序排列
- 通过右上角的排序选择器切换

### 操作

- **进入详情**：点击收藏卡片 → 进入影片详情页 → 选集播放
- **取消收藏**：右键点击收藏卡片 → 选择「取消收藏」

### 如何收藏影片

在影片详情页，点击元数据徽章行中的 **❤️ 收藏** 按钮即可。收藏后可通过 iCloud 同步到其他设备。

```
我的收藏 → 点击收藏影片 → 详情页 → 选集播放
```

## 设置

通过菜单栏 → Lumen → 设置（`⌘ ,`）打开设置窗口。

### 通用设置

| 选项 | 说明 | 默认值 |
|------|------|--------|
| 默认倍速 | 播放时的默认播放速度 | 1.0x |
| 快进/快退步长 | 方向键快进快退的秒数 | 10 秒 |
| 自动播放下一集 | 播完一集后自动播放下一集 | 开启 |
| 外观模式 | 跟随系统 / 深色 / 浅色 | 跟随系统 |

### 视频源管理

视频源标签页提供完整的源管理功能：

| 操作 | 说明 |
|------|------|
| ➕ **新增** | 通过 URL 或口令添加新的视频源 |
| ✏️ **编辑** | 修改源的名称或 URL |
| 🗑️ **删除** | 删除视频源及其所有关联站点 |
| 🔄 **检测** | 检测源内各站点的可用性 |
| 🔒 **加密** | 为视频源设置密码保护 |
| 🔛 **启用/禁用** | 切换源或站点的启用状态 |

每个源展开后会显示其包含的所有站点列表，包括：
- 站点名称、优先级、类型（插件/直连/主题）
- 检测状态（绿/红指示灯）
- 备注信息
- 单站点启用/禁用开关

### 高级设置

高级设置提供细粒度的缓存时间控制：

| 缓存类型 | 说明 | 默认值 |
|----------|------|--------|
| 主页缓存 | 主页内容缓存时长 | 4 小时 |
| 主页焦点图缓存 | Hero Banner 缓存时长 | 48 小时 |
| 分类列表缓存 | 分类列表缓存时长 | 7 天 |
| 分类内容缓存 | 分类内的影片列表缓存 | 4 小时 |
| 详情页缓存 | 影片详情页缓存 | 1 小时 |
| 完结内容缓存 | 已完结影片的缓存 | 7 天 |
| 插件引擎缓存 | 插件脚本缓存 | 4 小时 |

- 点击 **「清除全部缓存」** 可释放所有缓存占用的存储空间
- 缓存时间修改仅对新写入的缓存生效

## 键盘快捷键

| 快捷键 | 功能 |
|--------|------|
| `⌘ ,` | 打开设置 |
| `空格` | 播放 / 暂停 |
| `←` `→` | 快退 / 快进（默认 10 秒） |
| `回车` | 发起搜索 |
| `Esc` | 退出播放器 |
